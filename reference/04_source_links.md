## Review Cadence

Before using code directly, re-check:

- license
- last commit and release activity
- supported Python/runtime versions
- dependency health
- issue activity
- broker/API compatibility
- whether examples use point-in-time data
- whether backtests include costs, slippage, and realistic calendars
